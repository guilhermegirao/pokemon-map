# pokemon-map
Um simples sistema de mapa de andar baseado em Pokémon.

## Features
- Sistema de Andar pelo mapa
- Sistema de Colisão
- Sistema de Câmera
- Sistema de NPC e NPC Walk
- Suporte a múltiplos jogares (Ainda não incrementado)
- Suporte a vários mapas
